recode_file

recode_file -- Recode from file to file according to recode request.

Description

bool recode_file(int input, int output);

Recode the file referenced by file handle input into the file referenced by file handle output according to the recode request request. Returns FALSE, if unable to comply, TRUE otherwise.

This function does not currently process filehandles referencing remote files (URLs). Both filehandles must refer to local files.
